## 1) Ders Notları
- SPSS 2 farklı arayüz bulunur: 
    - Variable view: Özelliklerin girildiği arayüz
    - Data view: Verilerin girildiği arayüz
- Syntax kısmında SPSS içerisinde kod yazılabilir.
- Outputs kısmından çıktılar görülebilir, kaydededilebilir ve farklı formatlarda dışarı aktarılabilir.
- Variable View arayüzünde values değeri verinin etiketlenmesi için kullanılır. Örneğin Kadın=1 etiketiyle 2=Erkek etiketiyle belirtilebilir.
- Missing özelliği analizde kullanılmayan sınıfları göz ardı etmek için kullanılır.

- Excel dosyasını SPSS'e aktarmak için:
    - File
    - Open
    - Data
    - Dosyayı Seç
    - Dosyayı incele ve Ok ile aç

- SPSS'den dosyayı aktarmak için:
    - File
    - Export
    - Aktarmak istediğin data türünü seç
    - Save e bas ve kaydet
- Herhangi bir grafik çıkartmak için Analyze/Descriptive Statistics/Frequencies kısmından yapabiliriz.
- Sadece sayısal değişkenler için sayısal dağılımına bakmak için Analyze/Descriptive Statistics/Descriptives ile yapabiliriz.
